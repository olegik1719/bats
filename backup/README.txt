Скрипт Бэкапа.

Список необходимых фалов в папке:

1. Сам скрипт. 
	- backup.bat -- Его имя должно совпадать с именем файла настроек.(см. ниже)
	
	Также, имя скрипта влияет на имена логов этого скрипта.

2. Файл настроек.
	- backup.ini -- Его имя должно совпадать с именем файла bat.(см. выше)
	Синтаксис файла настроек:
	Настройка=Значение
	# Комментарий
	Образец файла настроек -- backup.ini
	
3. mykey.ppk -- приватный ключ, от зарегистрированного на сервере. Генерируется с помощью PUTTYGEN  и регистрируется на сервере.
	
4. Архиватор, с помощью которого сжимается папка
	7za -- 7-zip, standalone console version, можно скачать с 7-zip.org (не требует установки!)
	
5. Утилиты для работы с ssh:
	PSCP -- копирование файлов по протоколу scp
	PLINK -- выполнение команд через ssh
	взяты из набора putty. https://www.chiark.greenend.org.uk/~sgtatham/putty/latest.html
	Для запуска в авто режиме из-под пользователя, надо из-под него хотя бы раз этот скрипт запустить, чтобы прописать Фингерпринт в реестре (ответить Yes).

6. Отправка писем из консоли по ssl/tls:
	mailsend -- собственно, отправка.
	https://github.com/muquit/mailsend/releases/
	Скачать и переименовать(убрать версию в конце имени файла). Файл должен быть назван mailsend.exe.
	

	
Допустимо расположение нескольких копий скрипта в одной папке, для увеличения объема работы. Но у каждой копии должен быть свой файл настроек.
